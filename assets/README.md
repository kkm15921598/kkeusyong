# assets — 캐릭터 이미지 & 화면 시안

여기에 캐릭터 이미지와 홈 화면 시안을 올려주세요.
파일을 올린 뒤 경로만 알려주시면 코드(`app.js` / `styles.css`)에 연결하겠습니다.

## 폴더 구조
```
assets/
├── characters/   ← 사용시간 5단계 캐릭터 이미지
└── mockups/      ← 홈 화면 등 디자인 시안
```

## 캐릭터 권장 파일명 (사용시간 5단계)
> 아래 이름으로 올려주시면 단계별로 자동 매칭하기 편합니다. (png 권장, 정사각형/투명배경이면 더 좋음)

| 단계 | 사용시간 | 상태명 | 권장 파일명 |
|---|---|---|---|
| 1 | 0 ~ 2시간 미만 | 아주 좋은 흐름 | `char-1-good.png` |
| 2 | 2 ~ 4시간 미만 | 괜찮은 편 | `char-2-okay.png` |
| 3 | 4 ~ 5시간 미만 | 주의 필요 | `char-3-warning.png` |
| 4 | 5 ~ 7시간 미만 | 사용시간 많음 | `char-4-heavy.png` |
| 5 | 7시간 이상 | 디톡스 필요 | `char-5-detox.png` |

> 파일명이 달라도 괜찮습니다. 올려두고 "1단계는 OO.png" 식으로만 알려주세요.
